# 📺 YouTube Revenue Predictor - Machine Learning Project

## 📌 Project Overview

YouTube Revenue Predictor is a Machine Learning-based project designed to estimate the potential ad revenue generated by a YouTube video using performance-related data.

The system analyzes important video metrics such as views, likes, dislikes, comments, and video duration to identify patterns between audience engagement and revenue generation.

This project applies Artificial Intelligence and Data Science techniques to build a predictive model that helps estimate revenue based on historical YouTube data.

---

## 🎯 Problem Statement

Content creators and businesses often want to estimate the earning potential of videos before investing time and resources into content creation.

Since YouTube revenue depends on multiple factors such as engagement, views, and video characteristics, predicting revenue manually is difficult.

This project solves this problem by using Machine Learning algorithms to analyze historical video data and predict expected ad revenue automatically.

## ⚙️ How It Works

The project follows a complete Machine Learning workflow:

### 1. Data Collection

The system uses YouTube video datasets containing information about video performance and engagement.

---

### 2. Data Preprocessing

The dataset is prepared by:

* Removing missing values
* Cleaning unnecessary columns
* Converting video duration into numerical format
* Encoding categorical data
* Preparing features for model training

---

### 3. Feature Selection

The model uses important factors such as:

* Number of views
* Number of likes
* Number of dislikes
* Number of comments
* Video duration

to predict revenue.

---

### 4. Model Training

A **Linear Regression model** is trained using historical YouTube data to learn relationships between video performance and revenue.

---

### 5. Prediction

Users can enter video statistics, and the trained model predicts the estimated ad revenue.
